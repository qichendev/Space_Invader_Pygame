## Project Description

A player-controlled spaceship, enemies, bullets, and a real-time scoring system are all included in this Python-based Space Invaders game created with Pygame. GitHub version control, automated testing, a CI/CD pipeline, and deployment to an Azure virtual machine include a few of the ways the application showcases DevOps techniques. It satisfies every course requirement by applying cloud deployment, teamwork, and interactive gameplay.
